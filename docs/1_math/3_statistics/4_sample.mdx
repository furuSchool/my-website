## 母集団と標本
:::tip[母集団と標本]
母集団の特性を標本を調査することにより推測することが目的。この行為を **統計的推測** と呼ぶ。
- **母集団** : 調査対象となる全ての個体の集合
- **標本** : 母集団から無作為に選ばれた個体の集合。
    - 母集団から標本を選ぶことを **標本抽出** と呼ぶ
- 母集団の大きさを $N$ ($\infty$ も含む) ，標本の大きさを $n$ とし，母集団分布 $F$ の累積分布関数 $F(x)$ とおく。

ここで，$F$ から復元単純無作為抽出を行うことは，$F$ から独立な $n$ 個の確率変数 $X_1, \cdots, X_n$ を得ることと同じである。
$X_i$ が互いに独立に分布 $F$ に従う時，$X_1, \cdots, X_n$ は **独立同一分布 (i.i.d.)** であるという。
:::
- $X_1, \cdots, X_n$ が i.i.d. であるには，復元標本抽出を無作為に行うことが必要である
- $X_1, \cdots, X_n$ を母集団分布 $F$ から独立に抽出した大きさ $n$ の標本であると呼ぶこともある
- 標本抽出には二種類ある
    1. **復元抽出** : 抽出した個体を元に戻してから次の個体を抽出する方法
    2. **非復元抽出** : 抽出した個体を元に戻さない方法

:::note[統計モデル]
特に，母集団分布 $F$ が特定の分布族に属すると仮定することが多い。これを **統計モデル** と呼ぶ。
:::

### 統計量
:::tip[統計量]
標本 $X_1, \cdots, X_n$ を実数値関数 $T(X_1, \cdots, X_n)$ によって変換したものを **統計量** と呼ぶ。また，統計量の分布を **標本分布** と呼ぶ。

<span style={{backgroundColor : "yellow", fontSize : "120%"}}>**統計量は，標本によって決まる確率変数である**。</span>
以下は統計量の例。
- 標本平均 : $\bar{X} = \frac{1}{n} \sum_{i=1}^n X_i$
- 分散 : $s^2 = \frac{1}{n} \sum_{i=1}^n (X_i - \bar{X})^2$
:::
