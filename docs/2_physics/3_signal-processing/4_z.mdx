- 特定のフィルタを作ったり，逆に入力と出力の関係からシステムの特性を求めたり $\to$  $z$ 変換が便利
- ラプラス変換離散版

:::tip[ $z$ 変換]
**$z$ 変換：離散信号のシステムを表現するための手法**

片側 $z$ 変換：$X(z) = \sum_{n=0}^{\infty} x(n) z^{-n}$ （因果性：$x(n) = 0~(n < 0)$が成立する時）  
両側 $z$ 変換：$X(z) = \sum_{n=-\infty}^{\infty} x(n) z^{-n}$  
逆 $z$ 変換：$x(n) = \frac{1}{2\pi j} \oint_C X(z) z^{n-1} dz$
:::
信号サンプル値列：$x^*(t) = \sum_{n=0}^{\infty} x(nT) \delta(t - nT)$ をラプラス変換すると， $X^*(s) = \sum_{n=0}^{\infty} x(nT) e^{-snT}$ となるので，
$z = e^{sT} = e^{(\sigma + j \omega) T}$ と考えられる。

これにより，離散信号のシステムの安定性や特性を解析できるようになる。主に，**差分方程式を解く際**に使われる。
- $x(n)$ の差分方程式を $z$ 変換すると，$X(z)$ の代数方程式に変換される。特に，$x(n-1) \leftrightarrow z^{-1}X(z)$ である

---

##  $z$ 変換の性質
- 片側 $z$ 変換：$\exists A > 0,~\exists \alpha,~|x_n| < A e^{-\alpha n}~(n \ge 0)~\Rightarrow~X(z)$ は $|z| > e^\alpha$ で収束
    - 別に単なる無限級数の収束条件と同じ = 収束半径の存在：収束領域を $|z| > r_0$ と表現できる
    - 因果的な信号 ($h(n) = 0~(n < 0)$) の時に使える
- 両側 $z$ 変換：$\sum_{n=0}^{\infty} x(n) z^{-n}$ と $\sum_{n=-\infty}^{-1} x(n) z^{-n}$ が共に収束すれば良い $\to$ ドーナツみたいになることも
    - ローラン展開みたいなイメージ，というかそのもの

> 1. 線形性：$a x(n) + b y(n)~\leftrightarrow~a X(z) + b Y(z)$
> 2. 時間の移動：$x(n-k)u(n-k)~\leftrightarrow~z^{-k}X(z)$
> 3. 離散畳み込み：$y(n) = h(n) \ast x(n) = \sum_{k=0}^{\infty}h(k)x(n-k)~\leftrightarrow~Y(z) = H(z)X(z)$  
（因果性：$h(n) = 0~(n < 0)$が成立する時）

他にもいろんな性質があるが，あまり使わないので省略。

### $z$ 領域と周波数領域の関係
$z = e^{sT} = e^{(\sigma + j \omega) T}$ なので，$z$ 領域の伝達関数 $H(z)$ の周波数伝達関数 $H(\omega) = H(z)|_{z = e^{j \omega T}}$ となる。
- $T$ は標本周期であり，ここから $H(\omega)$ は周期 $\omega = \frac{2\pi}{T}$ を持つ
- 別に，標本化したからそらそうだよねという話
- 正規化した周波数を利用して，$z = e^{j \Omega}$ とすることもある。
