## 電気回路の基本法則
:::note[物理量の定義]
#### 回路で定義される量
- 電圧，電位 ( 記号 $V$、単位 V ) : 静電場 $E$ の中で単位電荷を $a$ から $b$ へ移動させるのに必要な仕事量。$V_{ba} = \int_a^b E \cdot dl$
    - グランド ( $0 V$ ) を基準に，回路の任意の点で定義される
    - 素子の両端の電位差をその素子にかかる電圧と呼ぶこともある
- 電流 ( 記号 $I$、単位 A ) : 単位時間あたりにある断面を通過する電荷量
    - 回路の任意の動線で定義される
- 電力 ( 記号 $P$、単位 W ) : 単位時間あたりに消費されるエネルギー量。$P = VI$

#### (集中定数回路の)素子
- 抵抗 ( 記号 $R$、単位 $\Omega $) : $V = RI$ 
    - $V$ は抵抗にかかる電圧、$I$ は抵抗を流れる電流
    - コンダクタンス (記号 $G$、単位 S) は $G = \frac{1}{R}$
- インダクタ，コイル ( 記号 $L$ : インダクタンス、単位 H ) : $V = L \frac{dI}{dt}$
    - 電流の変化により磁束が変化した際に電圧が生じる
    - $V$ はコイルにかかる電圧、$I$ はコイルを流れる電流
- キャパシタ，コンデンサ ( 記号 $C$ : キャパシタンス、単位 F ) : $Q = CV,~V = \frac{1}{C} \int_{t_0}^t I dt$ (ただし， $Q(t_0) = 0$ )
    - 電流 $I$ が流れると電荷が蓄積される
    - $Q$ はコンデンサに蓄積される電荷量、$V$ はコンデンサにかかる電圧

### 電源
- 理想電圧源 : 電流 $I$ に無関係に電圧 $V$ が一定
    - 実際には，内部抵抗を直列に接続して考えるべき
- 理想電流源 : 電圧 $V$ に無関係に電流 $I$ が一定
    - 実際には，内部抵抗を並列に接続して考えるべき
:::

- 節点 : 回路の接続点
    - 節点方程式 : 節点に流れ込む電流の総和は $0$ である (KCL)
- 枝 : 節点を結ぶ導線や素子を表す。導線では電流，電圧が一定
- グラフ : 回路を節点と枝で表現したもの
- カットセット : グラフの枝の部分集合で，その枝を切断するとグラフが2つに分離するもの
    - カットセット方程式 : カットセットの枝電流の総和は $0$ である
- グラフの木 : 全ての接点を含み，閉路のない枝で連結されているグラフ
    - 木の枝数 = 節点数 - 1
- タイセット : 一つの閉路を成す枝の集合
    - タイセット方程式 : タイセットの枝電圧の総和は $0$ である (KVL)


### 基本法則
- 合成抵抗は，直列接続では $R = R_1 + R_2 + \cdots$、並列接続では $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots$

なお以下は，直流・交流回路の両方に適用される。

:::tip[キルヒホッフの法則]
#### KCL ( Kirchhoff's current law ) : 回路の任意の節点において，枝から流れ込む電流の総和は $0$ である。
- 電荷保存則を意味する。$\frac{\partial \rho(\bm{r}, t)}{\partial t} + \nabla \cdot \bm{i}(\bm{r}, t) = 0$ で $\frac{\partial \rho(\bm{r}, t)}{\partial t} = 0$
#### KVL ( Kirchhoff's voltage law ) : 回路の任意の閉路において，一周した時の素子にかかる電圧の総和は $0$ である。
- $\nabla \times \bm{E} = 0$ を意味する
:::

:::note[重ね合わせの理]
複数の電源がある場合，電源が 1 つずつしかない場合の回路を考え，その結果を足し合わせることで全体の回路を解くことができる。
<img src="https://detail-infomation.com/wp-content/uploads/2022/02/956528fe44c08335a9393eedc8d46050-700x683.png" alt="重ね合わせの理" style={{ width: "80%" }} />
図は [こちら](https://detail-infomation.com/superposition-theorem/) より引用
:::
すなわち，一つの電源以外の電流源を短絡，電圧源を開放して回路を解くことを繰り返し，結果を足し合わせる。

:::note[鳳・テブナンの定理，ノートンの定理]
ある電源を含む 2 端子回路について，シンプルな回路に変換することができる。
- 鳳・テブナンの定理 : 2 端子間を開放した時の電圧が $V_0$， 2 端子回路の電源を除去した時のインピーダンスが $Z_0$ の時，$Z_0$ と $V_0$ の直列接続に置き換えられる
- ノートンの定理 : 2 端子間を短絡した時の電流が $I_0$， 2 端子回路の電源を除去した時のアドミタンスが $Y_0$ の時，$Y_0$ と $I_0$ の並列接続に置き換えられる
<img src="https://detail-infomation.com/wp-content/uploads/2021/12/4aa9c2a33e4200f04b9004514941d3b6-1-700x576.png" alt="鳳・テブナンの定理とノートンの定理" style={{ width: "80%" }} />

図は [こちら](https://detail-infomation.com/difference-between-thevenin-and-norton/) より引用
:::
つまり，2 端子回路でごちゃごちゃしている部分は，電源一つとインピーダンス一つに置き換えることができる。
結果として， 2 端子間にインピーダンス $Z$ を繋ぐ場合，そこにかかる電圧 $V$ と流れる電流 $I$ は，  
$V = V_0, I = \frac{V_0}{Z_0 + Z}$ (鳳・テブナンの定理) または $V = \frac{I_0}{Y_0 + Y}, I = I_0$ (ノートンの定理) となる。
