title: 相対論の気持ち

ここでは, 相対論に関する内容をまとめています. が, **趣味レベルで自学したもの**になります.

そのため, 物理学的に明らかに不適当な表現や誤りがあるかもしれないです. ご了承ください. 
一応, 特殊相対性理論と一般相対性理論の両方をさらえればいいなと思っています.

参考文献：
- 広江 克彦. 趣味で相対論.
- 服部 誠. "相対論". https://www.astr.tohoku.ac.jp/~hattori/Relativity.pdf.
- 棚橋典大. "相対性理論　講義ノート". https://www2.yukawa.kyoto-u.ac.jp/~norihiro.tanahashi/pdf/SR/note_SR.pdf.
