## 基本
:::tip[通信の目的]
情報を効率的に，素早く，正確に伝達するための技術。
:::


## 通信の大まかな流れ
1. 送信者が情報を送信
2. 情報がアナログで通信がデジタルなら，A/D 変換
    - サンプリング，量子化，符号化
3. 情報源符号化：情報をどう簡潔にするか，どのように 0/1 に変換するか
    - シャノンの通信路符号化定理：$C=B\log_2(1+SNR)~bit/s$ 通信路における誤り率 0 とできる最大情報転送速度
    - ハフマン符号，Run-Length 符号化など
4. 通信路符号化：誤り訂正など
    - CRC（巡回冗長検査），畳み込み符号，LDPC（低密度パリティ検査符号）など
    - 信頼性 $\leftrightarrow$ 効率
5. 変調：通信路を通じて情報を送信しやすくするために，情報を別の信号に変換するプロセス
    - 振幅，周波数，位相のいずれかに乗せる
    - アナログ：AM, FM, PM
    - デジタル：ASK, PSK, FSK, QAM
6. チャネルに流す $\rightarrow$ ノイズや干渉が加わる
7. 復調：PCM, PAM など
8. 通信路復号
9. 情報源復号
10. （D/A 変換）
11. 受信者が情報を受信
