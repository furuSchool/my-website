## 縦波
長さ $l$，断面積 $S$，ヤング率 $E$，密度 $\rho$ の一様な直方体について， $x$ 軸方向の一様でない伸び $u(x,t)$ を考える。
この時，$x$ 軸と平行な面に働くその面に垂直な力を $\displaystyle F(x,t) = ES\frac{\partial u}{\partial x}$ となる。

また、運動方程式を考えると，$\displaystyle \rho \frac{\partial^2 u}{\partial t^2} = E \frac{\partial^2 u}{\partial x^2}$ となる。
これは **波動方程式** である。  
一般解は，$u(x,t) = f(x - ct) + g(x + ct)~(c = \sqrt{\frac{E}{\rho}})$ となる。
- 微小部分で，$\frac{F}{S} = E \frac{\Delta l}{l}$ と $m \frac{\partial^2 u}{\partial t^2} = F$ を用いた

液体や気体については，平面波（すなわち一次元的に伝わる波）を考えると，$\displaystyle -p(x,t) = K\frac{\partial u}{\partial x}$ ($K$ は体積弾性率) となる。
また，$\displaystyle \rho \frac{\partial^2 u}{\partial t^2} = K\frac{\partial^2 u}{\partial x^2}$ となり，速度 $c = \sqrt{\frac{K}{\rho}}$ となる。

さらに気体について別の表現をする。振動が断熱的に起きると仮定すると，$pV^\gamma = \text{const}$ となる。
すると，$\Delta p = -\gamma p \frac{\Delta V}{V}$ となるので，$K = \gamma p$ より，
$v = \sqrt{\frac{\gamma p}{\rho}} = \sqrt{\frac{\gamma RT}{M}}$ となる。
- ここから，音波(空気中の縦波)は $v \approx 331.45 + 0.607~T$ (m/s) であることがわかる。

## 横波
### ねじれ波
半径 $R$，ずれ弾性率 $G$，ねじれの角度 $\Phi(z,t)$ の円柱を考える。この時，微小部分に対して 
$I\frac{\partial^2 \Phi}{\partial t^2} = N$ を適用して整理すると，
$\displaystyle \rho \frac{\partial^2 \Phi}{\partial t^2} = G \frac{\partial^2 \Phi}{\partial z^2}$ となる。

この時，速度 $c = \sqrt{\frac{G}{\rho}}$ となり，半径にはよらない。

### 曲げの波
断面積 $S$ の真っ直ぐであった棒や板に非一様な曲げが生じている。位置 $x$ の断面にモーメント $M(x,t)$ が働き，$z$ 軸方向に $z(x,t)$ の変位があるとする。

接線応力 $f(x)$，密度 $\rho$ として，$\frac{\partial M}{\partial x} \Delta x =  M(x + \Delta x,t) - M(x,t) = \Delta x \cdot f(x)S$ より，
$M(x,t) = EI \frac{\partial^2 z}{\partial x^2}$ となる。
また，$z$ 軸方向の運動方程式 $\rho (S \Delta x) \frac{\partial^2 z}{\partial t^2} = -f(x + \Delta x)S + f(x)S$ なので，
$v = \sqrt{\frac{E}{\rho}},~\kappa = \frac{I}{S}$ とおいて整理して，

$\displaystyle \left(\frac{\partial}{\partial t} \pm i\kappa v \frac{\partial^2}{\partial x^2}\right)z(x,t) = 0$ となる。
これはシュレディンガー方程式と同じ形である。

### 参考
弾性体の境界条件には，次の三つがある。
1. 固定：端の位置 $z$ とその傾き $\frac{\partial z}{\partial x}$ が与えられ変化しない
2. 支持：端の位置 $z$ が与えられ，モーメント $M = 0$ となる
3. 自由：モーメント $M = 0~(\frac{\partial^2 z}{\partial x^2} = 0)$，応力 $f = 0~(\frac{\partial^3 z}{\partial x^3} = 0)$ となる
