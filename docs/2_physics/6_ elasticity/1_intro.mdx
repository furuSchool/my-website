## 連続体と弾性体と流体

:::tip[定義]
- **連続体**：物質の微小な部分を無限に分割しても，その性質が変わらない物体。
    - すなわち，物体の性質はその平均量で表現できる。
- **弾性体**：連続体のうち，外力を受けると変形するが，外力が取り除かれると元の形に戻る物体。
- **流体**：連続体のうち，自由に形を変えて流れることができる物体。
:::
連続体力学が取り扱える長さは原子間距離よりも十分大きい長さであり，時間間隔も，平均自由行程を進むよりも十分長い時間であると仮定している。

また，連続体は大きく分けると弾性体と流体だが、他にも、塑性体、粘弾性体、非ニュートン流体など様々に分類が可能である。(考える時間スケールや状況に依存する)

## 弾性体の変形
### 伸縮
長さ $l$，断面 $S$ の直方体の弾性体の両断面に伸びる方向に力 $F$ を加えると，その長さは $\Delta l$ だけ伸びたとする。

この時，**(法線)応力** $\displaystyle \sigma = \frac{F}{S}$ と定義し，十分伸びの割合が小さい時は，
**フックの法則**：$\displaystyle \sigma = E \varepsilon$ が成り立つ。

ここで，$\varepsilon = \frac{\Delta l}{l}$ は **ひずみ** と呼ばれ，$E$ は **ヤング率** と呼ばれる。

また，より一般に，応力と伸びの関係は，**応力-ひずみ曲線** として表される。
図は，[応力ｰひずみ曲線とは？応力とひずみの関係からわかることやグラフの見方](https://chansato.com/doboku/stress-strain-curve/) より引用。

<img src="/img/physics/stress-strain-curve.png" width="50%" alt="応力-ひずみ曲線"/>

- 比例限界：フックの法則が成り立つ限界
- 弾性限界：弾性体でなくなる限界
- 弾性限界：弾性体でなくなる限界
    - 弾性ヒステリシス：応力を加えるときと取り除くときで，応力-ひずみ曲線が異なる現象
- 降伏点：物体内部にすべりを生じ，応力をほとんど変化させなくてもひずみが増加し始める点
    - 上降伏点：応力が減少し始める点と，下降伏点：応力が増加し始める点，降伏棚：応力をほとんど変化させなくてもひずみが増加する領域，などがある場合も
- 引っ張り強さ：応力の最大値
    - 例：鋼だと数百 MPa 程，クモの糸は $1.8 \times 10^8$ Pa 程
- 破断点：物体が破断する点

### ポアソン比
伸縮させると，横方向にも変形が生じる。直方体の断面を $h \times w$ とし，横方向の変化を $\Delta h, \Delta w$ とすると，

$\displaystyle \frac{\Delta h}{h} = \frac{\Delta w}{w} = - \nu \frac{\Delta l}{l}$ が成り立つ。
ここで，$\nu$ は **ポアソン比** と呼ばれ，物体に固有の値である。
- 例：鋼だと $0.3$ 程，ゴムだと $0.5$ 弱

### 圧縮・膨張
体積 $V$ の弾性体に一様な圧力 $p$ を加えると，その体積が $V$ から $V + \Delta V$ に変化したとする。

この時，$\displaystyle \frac{\Delta V}{V} = - \kappa p,~ p = - K \frac{\Delta V}{V}$ が成り立つことが知られている。

ここで，$\kappa$ は **体積弾性率**，$K = \frac{1}{\kappa}$ は **体積弾性係数** と呼ばれる。

さらに，$K$ と $E$ の関係は，$\displaystyle K = \frac{E}{3(1-2\nu)}$ であることが計算するとわかる。（一応近似的な関係）
- 応力 $P$ が各 3 面にかかっているとしてそれぞれで計算すれば，例えば $\frac{\Delta l}{l} = -\frac{1}{E} p + \frac{\nu}{E} p + \frac{\nu}{E} p$ となる。

### ずれ
直方体の弾性体について，下面を固定し，面積 $S$ の上面に平行な力 $F$ を加えると，その面がずれる。（ "純粋なずれ" を引き起こすには，他の三面にもうまく力を加える必要がある）

この時，**せん断応力(接線応力)** $\displaystyle \sigma = \frac{F}{S}$ と定義し，十分小さいずれの割合が小さい時は，ずれの角度を $\theta$ として，

$\displaystyle \sigma = G \theta \approx G \frac{\Delta x}{h}$ が成り立つ。

ここで，$G$ は **ずれ弾性率(剛性率)** と呼ばれる。また，$\theta \approx  \frac{\Delta x}{h}$ である。( $h$ は上面と下面の距離)

さらに， $G$ と $E$ の関係は，$\displaystyle G = \frac{E}{2(1+\nu)}$ であることが計算するとわかる。
- 力の向きと大きさを整理して，考えやすい長方形をとって考えれば良い（図は[こちら](https://kenchik.info/2022/04/08/e_g/)より引用）
<img src="/img/physics/G-E.jpg" width="50%" alt="G と E の関係"/>

:::tip[まとめ]
- **(法線)応力** $\sigma = \frac{F}{S}$：考えている面 (面積 $S$ ) に垂直な力 (力 $F$ ) の単位面積あたりの大きさ

- **ひずみ** $\varepsilon = \frac{\Delta l}{l}$：考えている面に垂直な変位の大きさ

    - **フックの法則** $\sigma = E \varepsilon$ が成り立つ。$E$ は **ヤング率**

- **ポアソン比** $\nu$：横方向と伸びる方向の変化の比。$\frac{\Delta h}{h} = \frac{\Delta w}{w} = - \nu \frac{\Delta l}{l}$ が成り立つ

- **せん断応力(接線応力)** $\sigma = \frac{F}{S}$：考えている面 (面積 $S$ ) に平行な力 (力 $F$ ) の単位面積あたりの大きさ

    - $\sigma = G \theta$ が成り立ち，$G$ は **ずれ弾性率(剛性率)**，$\theta \approx  \frac{\Delta x}{h}$ はずれの角度である。
- **体積弾性率** $\kappa$：体積の変化と圧力の比。$\frac{\Delta V}{V} = - \kappa p$ が成り立つ
- **体積弾性係数** $K = \frac{1}{\kappa}$：$p = - K \frac{\Delta V}{V}$ が成り立つ
- $\displaystyle K = \frac{E}{3(1-2\nu)},~ G = \frac{E}{2(1+\nu)}$ などが成り立つ
（$E, \nu, G, K$ は互いに関係し合い，二つの値がわかれば他の値も計算できる）
