---
title: 6. 剛体の運動
---
これまでは，質点の運動について考えてきたが，次に剛体の運動について考える。

:::tip[剛体]
剛体とは，質量と大きさを持つが，どのような力を受けても変形しない物体のことである。
質点とは異なり，回転運動を考慮する必要がある。
:::
- 三次元空間の質点の自由度は 3 つであるが，剛体の自由度は 6 つである

---
## 基本
### 剛体の運動方程式
:::tip[剛体の運動方程式]
剛体の運動方程式は，以下のように表される。
- **並進運動：$\displaystyle \bm{F} = \frac{d\bm{P}}{dt}$**
- **回転運動：$\displaystyle \frac{d\bm{L}}{dt} = \bm{N}$**

ただし，全質量 $M = \int \rho(\bm{r}) dV$，質量中心 $\bm{R} = \frac{1}{M} \int \rho(\bm{r}) \bm{r} dV$，
全運動量 $\bm{P} = \int \rho(\bm{r}) \bm{v} dV$，全角運動量 $\bm{L} = \int \rho(\bm{r}) \bm{r} \times \bm{v} dV$，
外力 $\bm{F} = \sum_i \bm{F}_i$，外力モーメント $\bm{N} = \sum_i \bm{r}_i \times \bm{F}_i$ である。
:::
角運動量 $\bm{L} = \bm{R} \times M\bm{V} + \bm{L}_{\text{rel}}$ なので，重心周りの角運動量 $\bm{L}_{\text{rel}}$ を用いて，
$\frac{d\bm{L}_{\text{rel}}}{dt} = \bm{N}_{\text{rel}}$ と表すこともできる。  
また，並進運動は質量中心の位置 $\bm{R}$ に対する運動方程式を考えることで，剛体全体の運動を考えて，$\bm{F} = M\ddot{\bm{R}}$ と表すこともできる。

これらにより，剛体の運動は， $M$ の質点と同じ方程式で表される質量中心の運動と，その質量重心周りの回転運動の足し合わせで表される。

### 剛体のつり合い
:::tip[剛体のつり合い]
剛体がつり合う時，$\bm{F} = 0$ かつ $\bm{N} = 0$ である。ただし，以下の注意点がある。
- 剛体にはたらく力は，力の作用点を作用線に沿ってずらしても，力のモーメントが不変のためその作用は同じ
- どの点を基準にしても，力のモーメントの和はゼロである（ $\sum_i \bm{r}_i \times \bm{F}_i = \sum_i (\bm{r}_i + \bm{r}_0) \times \bm{F}_i = 0$ ）
- 質量中心と重心は一致する。従って，重力は質量中心にかかるとしてよい
- 偶力：合力はゼロだが，力のモーメントがゼロでない一対の力
:::

## 固定軸をもつ剛体の運動
まずは，空間に固定された軸周りの回転のみが可能な剛体の運動について考える。この時，自由度は 1 つである。

運動方程式は，固定回転軸を $z$ 軸として，$\displaystyle \dot{\bm{L_z}} = \bm{N_z}$ となる。（別に，$\bm{L_x} = 0,~\bm{L_y} = 0$ というわけではない）
### 慣性モーメント
この状況の時，円筒座標( $r, \theta, z$ )で考えると，$x = r\cos\theta,~y = r\sin\theta$ より，    
$\bm{L_z} = \int \rho(\bm{r})\bm{r} \times \bm{\dot{r}} dV = = \cdots = \int \rho(\bm{r}) r^2 \dot{\theta} dV$ となる。

ここで，$\theta$ は位置によらないため，$I_z = \int \rho(\bm{r}) r^2 dV$ として，$\bm{L_z} = I_z \dot{\theta}$ と表すことができる。

:::tip[慣性モーメント]
固定回転軸を $z$ 軸として，慣性モーメント $\displaystyle I = \int \rho(\bm{r}) r^2 dV$ と定義する。この時，以下が成り立つ。
- 慣性モーメント：$\displaystyle I = \int \rho(\bm{r}) r^2 dV$ $~~\leftrightarrow~~$ 慣性質量：$\displaystyle m$
- 運動方程式：$\displaystyle I \ddot{\theta} = N_z$ $~~\leftrightarrow~~$ 運動方程式：$\displaystyle m \ddot{\bm{r}} = \bm{F}$
- 角速度：$\displaystyle \omega = \dot{\theta}$ $~~\leftrightarrow~~$ 速度：$\displaystyle \bm{v} = \dot{\bm{r}}$
- 角運動量：$\displaystyle \bm{L_z} = I \omega$ $~~\leftrightarrow~~$ 運動量：$\displaystyle \bm{P} = m\bm{v}$
- 回転運動エネルギー：$\displaystyle K = \int \frac{1}{2} \rho(\bm{r}) \dot{\bm{r}}^2 dV = \frac{1}{2} I \omega^2$
$~~\leftrightarrow~~$ 運動エネルギー：$\displaystyle K = \frac{1}{2} m v^2$
:::
- 固定回転軸が $z$ 軸の時，$\dot{\bm{r}} = \dot{r}\bm{e}_r + r\dot{\theta}\bm{e}_\theta$ となる
- **回転半径** $k$：$Mk^2 = I$ となるような $k$ のこと。回転半径は剛体の形と軸の位置だけで決まり，
全質量が軸から $k$ だけ離れた位置に集中している時の慣性モーメントと同じ値を持つ

#### 平行軸の定理
：質量 $M$ の剛体の任意の軸周りの慣性モーメント $I$ は，質量中心を通りその軸に平行な軸周りの慣性モーメント $I_G$，二つの軸間の距離を $h$ として，
$\displaystyle I = I_G + Mh^2$ と表される。

#### 平板の慣性モーメント
：平板内に $x$ 軸，$y$ 軸をとった時，平板と垂直な $z$ 軸周りの慣性モーメントは，$I_z = I_x + I_y$ となる。

### 慣性モーメントの例
- 長さ $l$，質量 $M$ の細い棒の，中心を通り棒に垂直な軸周りの慣性モーメント：$I = \frac{1}{12} Ml^2$
    - 棒の端を通り棒に垂直な軸周りの慣性モーメント：$I = \frac{1}{3} Ml^2$
- 2 辺の長さが $a, b$ であり，質量 $M$ の長方形板の，板の中心を通り板に垂直な軸周りの慣性モーメント：$I = \frac{1}{12} M(a^2 + b^2)$
    - 板の中心を通り板に辺に平行な軸周りの慣性モーメント：$I = \frac{1}{3} Ma^2,~\frac{1}{3} Mb^2$
- 3 辺の長さが $a, b, c$ であり，質量 $M$ の直方体の，中心を通り直方体の中心を通り各辺に平行な軸周りの慣性モーメント：  
$I = \frac{1}{12} M(a^2 + b^2),~\frac{1}{12} M(b^2 + c^2),~\frac{1}{12} M(c^2 + a^2)$
- 半径 $a$，質量 $M$ の薄い円盤の，円盤の中心を通り円盤に垂直な軸周りの慣性モーメント：$I = \frac{1}{2} Ma^2$
    - 円盤の中心を通り円盤に平行な軸周りの慣性モーメント：$I = \frac{1}{4} Ma^2$
- 半径 $a$ で質量 $M$ の細い線でできた円形の輪の，輪の中心を通り輪の軸周りの慣性モーメント：$I = Ma^2$
    - 輪の中心を通り輪の円周に平行な軸周りの慣性モーメント：$I = \frac{1}{2} Ma^2$
- 質量 $M$，半径 $a$，高さ $l$ の円柱の，中心を通り円柱の軸周りの慣性モーメント：$I = \frac{1}{2} Ma^2$
    - 円柱の中心を通り円柱の底面に平行な軸周りの慣性モーメント：$I = \frac{1}{4} Ma^2 + \frac{1}{12} Ml^2$
- 質量 $M$，半径 $a$ の球の，中心を通る軸周りの慣性モーメント：$I = \frac{2}{5} Ma^2$
- 質量 $M$，半径 $a$ の球殻の，中心を通る軸周りの慣性モーメント：$I = \frac{2}{3} Ma^2$

## 剛体の運動の例
### 実体振り子
水平な固定軸を $z$ 軸に取り，鉛直面内に $x$ 軸を鉛直下向きに，$y$ 軸を水平に取る。
そして，$z$ 軸まわりで重力のみを受けて運動する剛体を考える。

この時，運動方程式：$\displaystyle I\ddot{\theta} = \bm{r_G} \times M\bm{g} = -Mgh\sin\theta$ となる。
ただし，円筒座標( $r, \theta, z$ )で考えており，$h$ は重心から軸までの距離である。  
振幅が小さい時，$\sin\theta \fallingdotseq \theta$ として，整理すると，$\theta(t) = \theta_0\cos(\omega t + \alpha),~\omega = \sqrt{\frac{Mgh}{I}}$ となる。  
従って，振り子の周期は，$T = 2\pi\sqrt{\frac{I}{Mgh}} = 2\pi\frac{k_z}{\sqrt{gh}}$ ( $k_z$ は振り子の回転半径 ) とわかる。

また，長さ $l$ の単振り子の周期 $T = 2\pi\sqrt{\frac{l}{g}}$ と比較して，
実体振り子の **相当単振り子の長さ** $l_E = \frac{I}{Mh} = \frac{k_z^2}{h}$ と定義する。
- 例えば，剛体が半径 $a$ の円盤である場合，$I = \frac{1}{2} Ma^2 + Ma^2$ なので，$\ddot{\theta} = -\frac{2g}{3a}\sin\theta$ となる。
すなわち，長さ $a$ の単振り子と比べると，角振動数が $\sqrt{\frac{2}{3}}$ 倍になっており，$l_E = \frac{3}{2} a$ となる。これは質量中心周りの回転運動によるものである
    - 角振動数が最大となるのは，重心から軸までの距離 $b$ として，$I = \frac{1}{2} Ma^2 + Mb^2$ より，
    $\omega = \sqrt{\frac{Mgb}{\frac{1}{2} Ma^2 + Mb^2}}$ なので，$b = \frac{a}{\sqrt{2}}$ の時最大値 $\omega = \sqrt{\frac{g}{\sqrt{2}a}}$ となる
- 長さ $l$ の直方体の場合は，十分細長い場合，$l_E = \frac{2}{3} l$ となる

### 剛体の転がり運動
<img src="/img/physics/rolling.png" style={{ width: '20%'}} alt="剛体の転がり運動"/>
回転体の形をした半径 $a$，質量 $M$，重心周りの慣性モーメント $I$ の剛体が，角度 $\theta$ の斜面を転がり下る運動を考える。斜面と水平で下る向きを正として $x$ 軸にとり，
初期状態からの物体の回転角を $\phi$ とする。この時，以下の式が成り立つ。

$\displaystyle \begin{cases} M\ddot{x} = Mg\sin\theta - F \\ M\ddot{y} = -Mg\cos\theta + N 
\\ I\ddot{\phi} = Fa \end{cases}$

#### 1. 滑る場合 ( $F = \mu' N$ )
整理すると，$N = Mg\cos\theta,~\ddot{x} = g\sin\theta - \mu' g\cos\theta,~\ddot{\phi} = \frac{\mu^{\prime} a}{k^2} g\cos\theta$ となる。
また，$a\phi$ だけ転がり，$x - a\phi$ だけ滑る運動をする。この時，摩擦力の仕事 $\mu' Mg\cos\theta(x-a\phi)$ 分だけエネルギーが消費される。
- 滑らずに転がる場合に比べると，滑る場合のほうが並進運動の加速度は大きく，速く斜面を下る

#### 2. 滑らず転がる場合 ( $F < \mu N$ )
この時，拘束条件として，$x = a\phi$ が成り立つ。この時，運動方程式を整理すると，  
$\displaystyle(Ma^2 + I)\ddot{\phi} = Mga\sin\theta$ より，$\displaystyle\ddot{x} = \frac{1}{1 + \frac{k^2}{a^2}} g\sin\theta$ となる。

従って，転がる場合の実質的な重力加速度は，$g' = \frac{g}{1 + \frac{k^2}{a^2}}$ であり，
これは，同じ半径・質量でも **回転半径が小さい（慣性モーメントが小さい）物体ほど転がるのが早い** ことを示している。
- 早く転がる順は，滑る > 球 > 円柱 > 球殻 > 円筒 のようになる。ダンベル型の物体が転がる中では最も早いらしい

また，滑らないための条件は，$F < \mu Mg\cos\theta$ であり，整理して， $\tan\theta < (1 + \frac{a^2}{k^2})\mu$ となる。
これは，同じ半径・質量でも，**回転半径が大きい（慣性モーメントが大きい）ほど滑りやすい** ことを示している。

また，この時，力学的エネルギー $E = \frac{1}{2} M\dot{x}^2 + \frac{1}{2} I\dot{\phi}^2 - Mgx\sin\theta$ は保存する。（微分すればわかる）
- 斜面を転がる運動では，位置エネルギーが並進運動と回転運動に分配される。慣性モーメントが小さいほど回転運動に必要なエネルギーが少なくなるため，転がるのが早くなる

### 衝撃の中心
棒に，棒の長さ方向と垂直な向きに撃力を加える。重心 $G$ から距離 $b$ 離れた位置に撃力 $F$ を加えたとする。重心周りの慣性モーメントを $I_G$ とし，
撃力を加えた後の角速度 $\omega$，速さ $v_G$ とすると，以下の式が成り立つ。

$\begin{cases} Mv_G = \int F dt = p \\ I_G\omega = \int bF dt = bp\end{cases} $

ここで，撃力を加えた時に速度が $0$ となるような点と重心の距離を $h$ とする（重心に関して撃力が加えられた点と反対側にある）。
この時，$v_h = \frac{p}{M} - h \omega,~v_G = \frac{p}{M},~v_b = \frac{p}{M} + b \omega$ となる。

従って整理すると，$h = \frac{I_G}{Mb} = \frac{k_G^2}{b}$ となり，この点では撃力を感じない。
逆に，先に $h$ を決めた時に，撃力を受けたとしてもその衝撃を感じない点は，$b = \frac{k_G^2}{h}$ となる点であり，この点を **衝撃の中心** という。

- 重心から $h$ の場所をグリップとした時，どこにボールを当てれば衝撃を感じないかを考えているイメージ。衝撃の中心 = "芯"
- 例えば，半径 $a$ の球に高さ $h$ から撃力を加えると，ボールが初めから転がる，すなわち地面との設置面が衝撃を受けないような時，
$h = a + \frac{\frac{2}{5} Ma^2}{Ma} = \frac{7}{5} a$ となる。
