## 人の視覚
### 眼の構造

- 角膜：集光する。レンズのような働き
- 水晶体：厚さや曲率を変えて焦点距離を調整する。ピントを合わせる
- 虹彩：瞳孔の大きさを調整する。カメラの絞りに相当
- 中心窩：視力が最も高い部分。錐体細胞が集中している
- 網膜：光を受け取る。その下にある視神経が光を電気信号に変換して脳に送る

### 人の視覚特性
視細胞には，明視細胞（錐体細胞）と暗視細胞（桿体細胞）がある。
- 錐体細胞：色を感じる。L 錐体（赤），M 錐体（緑），S 錐体（青）に分かれる。錐体細胞による視覚を明所視という
- 桿体細胞：明暗を感じる。周辺部に多く分布する。光感度が高く暗いところで活動し，桿体細胞による視覚を暗所視という

視覚に関連する概念は他にも下記のようなものがあるが，詳細は省略する。
- 暗順応，明順応，コントラスト，空間周波数，フリッカー（時間周波数），エッジの知覚，線分サイズと傾き，奥行き知覚，両眼視差，運動視差，単眼の奥行き知覚，錯視・錯覚

## 色空間
### 加法混色（RGB）と減法混色（CMY）
:::tip[加法混色（RGB）と減法混色（CMY）]
- 加法混色：原色光である「赤(R)・緑(G)・青(B)」の光を混ぜて人間の目に色刺激を起こす
- 減法混色：白色光源下で，シアン(C フィルタ，R 吸収)・マゼンタ(M フィルタ，G 吸収)・イエロー(Y フィルタ，B 吸収)によって光を吸収して，
残りの反射された光により人間の目に色刺激を起こす
:::
- RGB は人間が視認できる全ての色を表現できるわけではない

### CIE-RGB 色空間
任意の色を，原刺激（赤・緑・青）の光源を混合することで実現する。
- 人間は，赤・緑・青の光を混ぜたものと対象からの反射光を区別できない。同じ色に見えるようにすることを，「等色」という。
- 赤(R)，緑(G)，青(B)のの等色関数 $\bar{r}(\lambda), \bar{g}(\lambda), \bar{b}(\lambda)$ 
に定数値 $\bm{R}, \bm{G}, \bm{B}$ を掛けたものを加算することで，任意の色を表現できる
    - $\bm{U}(\lambda) = \bar{r}(\lambda) \bm{R} + \bar{g}(\lambda) \bm{G} + \bar{b}(\lambda) \bm{B}$。ただし，$=$ は等色，$+$ は混色
    - 赤(R)の等色関数には負の値が含まれるため，単純な加算ではできない

### CIE-XYZ 色空間
等色関数が非負の値を取るように定義された色空間。$R, G, B$ から $X, Y, Z$ への線形変換で表現される。
- $Y$ は輝度を表し，人間の視覚の標準視感度に基づいて定義されている

### sRGB 色空間
デバイス独立色の一つ。デバイスの特性によって色が異なるため，色の再現性を確保するために作成された標準の RGB 色空間。
- 一般に使われているモニタで対応できるように，CIE-XYZ 色空間を非線形変換している

### CIE-L* a* b* 表色系
輝度，色度の変化をより人間の視覚に近づけた CIE 均等色空間。色相，彩度，明度を表現する。計量的な用途に使う。
- 人間の目で見える全ての色をほぼ表現できる

### マンセル表色系
心理学的な側面から，色相，明度，彩度の 3 属性で色を定義する。人間が直感的に使う。
- 色相：色相環を 5 等分し，R, Y, G, B, P を 5 つの基本色相とし。等分割して全体で 100 色相
- 明度：色相の明るさ。黒(0)から白(10)までの 11 段階
- 彩度：各色相・彩度ごとに，無彩色から最も鮮やかな色まで等分で区切る。

### HSI(HSV) 表色系
マンセル表色系を模した色空間。色相，彩度，明度のうちどれか 1 つだけを変化させたい時に使う。
- HSI 変換：RGB 空間から HSI 空間への変換。例えば，RGB だと明度を変えると色も濃くなってしまう

### YCbCr 表色系
画像・映像処理やデジタルビデオの圧縮で使われる色空間。輝度信号 $Y$ と色差信号 $Cb, Cr$ に分解する。
- 輝度信号 $Y = 0.299R + 0.587G + 0.114B$：人間は輝度を重視する
- 色差信号 $Cb = -0.1687R - 0.3313G + 0.5B, Cr = 0.5R - 0.4187G - 0.0813B$
- JPEG や 動画圧縮で使われる。人間は色差の方が鈍感

## 画像の特徴量
### 画像のノイズ
- ショットノイズ：到達する光子数の変動による，受光素子から読み出す電圧値に生じるゆらぎ
    - 到達する光子数はポアソン分布に従う
    - 信号が大きいほど標準偏差は大きいが，SNRは小さくなる

#### コントラスト：画像のヒストグラムの広がり = 画素値の広がり

#### シャープネス：画像の輪郭が鮮明に見える度合い
1. 輪郭(エッジ)部分の画素値変化を測定
2. 空間周波数に対する振幅の変化を測定

### トーンカーブ
:::tip[トーンカーブ]
入力画像の画素値を変換して出力画像の画素値を決定する関数
:::

#### γ補正： 出力画素値 $y = 255 (\frac{x}{255})^{\gamma}$ とする
- $\gamma < 1$：暗い部分のコントラストを上げる
- ブラウン管（CRT）の入出力特性を補正するために使われることが多い
