## 測定

:::tip[測定による状態の変化]
量子力学では測定の影響を取り除くことは原理的に不可能。
:::

### von Neumann の射影測定
:::tip[von Neumann の射影測定]
測定前の状態 $\ket{\psi}~\rightarrow~$ 測定後の状態 $\ket{\psi^{\prime}} = \frac{P_a\ket{\psi}}{|| P_a\ket{\psi} ||}$
-   いろんな可能性があった状態ベクトルから一つの固有ベクトルが選ばれたイメージ
:::
-   特に縮退がある場合は Lüders の射影測定という

より一般化した測定の枠組みがいくつか存在する

### PVM（projection-valued measure）測定
:::tip[PVM測定]
物理量 $A$ に対応する射影演算子 $P_a$ の集合 $\{P_a\}$ による測定
:::
- 物理量 $A$ の代わりに，射影演算子 $P_a$ の組 $\{P_a\}$ を考える
- こうすれば，測定値 $a$ が実数でない場合でも対応できる。測定値はシンボルだろうがベクトルだろうがなんでも良い
- ただし，$\sum_a P_a = I$ が成り立つ必要がある

ただし，PVM 測定では間接測定はできない
- 間接測定：ある注目する系を別の補助系と相互作用させ，補助系の物理量を測定することで注目する系の状態を調べるような測定

### POVM（positive operator-valued measure）測定
測定 $M$ について測定値の集合 $M = \{m_1, m_2, \ldots\}$ とする時，測定とはそれぞれの測定値の確率分布を定めることと考えられる。  
この時，一般の測定では，アフィン性：$p(m_1) + p(m_2) = p(m_1 + m_2)$ が成り立つ：これが成り立てば一般の測定を記述できるのでは？
- 以下は，密度演算子（5章）を先に知っていると理解しやすい

:::tip[POVM測定]
アフィン性を満たす測定。
エルミート空間 $\mathscr{H}$ 上の演算子の組 $\{E_m\}$ が，$E_m \geq 0$ かつ $\sum_m E_m = I$ を満たす時，$\{E_m\}$ は POVM と呼ばれる。

この時，測定 $M$ が POVM 測定：  
POVM $\{E_m\}$ を用いて，測定値 $m$ が得られる確率は $\text{Pr}(M = m | \rho) = \text{Tr}(E_m \rho)$
- $\rho$：密度演算子
- POVM の要素がすべて射影演算子の場合，POVM は PVM となる：$E_m E_n = \delta_{mn} E_m$
:::

一方で，間接測定を定式化すると下記のようになる
:::tip[間接測定]
1. 対象系 $S$ の初期状態を $\rho$，補助系 $A$ の初期状態を $\sigma$ とする。
    - $A$ と $S$ がはじめ相関がなければ，全体系は $\rho \otimes \sigma$ で表される
2. 全体系に対してユニタリ演算子 $U$ が作用して，$U(\rho \otimes \sigma)U^\dagger$ になる。
3. 補助系 $A$ の物理量 $B = \sum_m m P_m$ を測定する。  
この時，測定値 $m$ が得られる確率：
$\text{Pr}(B = m) = \text{Tr}_{SA}((I \otimes P_m)U(\rho \otimes \sigma)U^\dagger)$
:::
この時，$E_m = \text{Tr}_A((I \otimes \sigma)U^\dagger (I \otimes P_m)U)$ とすると，$E_m$ は POVM となる
- 実は，間接測定 $\Leftrightarrow$ POVM 測定
