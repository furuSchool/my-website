:::warning[目標]
まずは，量子力学を公理によって特徴づけるが，その前に必要な知識を整理する
- 通常の量子力学入門ではSchrödinger方程式から始まるが，量子情報においては時間発展より，**入力と出力関係**に重きが置かれる
- また，Schrödinger方程式で扱われる粒子のダイナミクスは量子情報では興味の対象ではない
:::

## 基本
:::tip[量子力学の基本]
- 量子力学は $\mathbb{C}^d$  上のHilbert空間 $\mathscr{H}$ で定義される
- 量子力学では，物理量や観測は Hilbert空間 $\mathscr{H}$ 上の線形演算子によって表される
    - 線形演算子とは，ベクトル空間上の線形写像のこと．行列と思えば良い
- 量子力学では，状態を Hilbert空間 $\mathscr{H}$ 上のケットベクトル $\ket{\psi}$ で表す
:::

また，下記のようなことも定義される
- $\psi(x) = \braket{x|\psi}$
    - ブラケット表現は，関数表現をより一般的な表現に拡張する．ベクトルと思えば良い
- ブラベクトル $\bra{\psi} = \ket{\psi}^\dagger$：ケットベクトルの Hermite 転置
- 内積：$\braket{\phi|\psi}$ 
- 外積：$\ket{\phi}\bra{\psi}$：結果は線形演算子．行列積と思えば良い


:::tip[いろんな定義]
- $A\ket{v} = v\ket{v}$ のとき，$\ket{v}$ は固有ベクトル，$v$ は固有値と呼ぶ（$A$：線型演算子）
- 線形演算子の対角化：$A = \sum_i \lambda_i \ket{i}\bra{i}$（$\lambda_i$：固有値，$\ket{i}$：固有ベクトルで正規直交基底）
    - なんか見慣れないね
    - 下記の射影演算子を使うと，$A = \sum_i \lambda_i P_i$ と書ける
- $A$ が正規：$A^\dagger A = AA^\dagger$
- $U$ が Unitary：$U^\dagger U = UU^\dagger = I$
    - Unitary演算子は内積を保存する：$\braket{U\psi|U\phi} = \braket{\psi|\phi}$
- $A$ が Hermite：$A = A^\dagger$
- $A$ が正定値：任意のベクトル（ $\ket{v}\neq 0$ ）に対して $\bra{v}A\ket{v}>0$
    - この時，$A$ は必ず Hermite である．また，$A$ の固有値は全て正である

:::
成り立つ定理は下記の通り
- スペクトル分解定理：$A$ が 正規 $\Rightarrow$ $A$ は対角化可能
- $A$ が Hermite $\Leftrightarrow$ $A$ の固有値が実数：$\bra{\psi}A\ket{\psi} \in \mathbb{R} ~ (\forall \ket{\psi})~\Leftrightarrow~A = A^\dagger$

### 射影演算子
射影演算子は，任意のベクトルをある部分空間に射影する．
:::tip[射影演算子]
$d$ 次元 Hilbert 空間 $\mathscr{H}$ の正規直交基底を $\ket{i}$ とし，$\ket{v}$ をベクトルとする．
- $\ket{v} = \sum_i v_i\ket{i} = \sum_i \braket{i|v}\ket{i}$：$\ket{v}$ は正規直交基底 $\ket{i}$ で表される
- $I = \sum_i \ket{i}\bra{i}$：正規直交基底だからね

この時，$\mathscr{H}$ の部分空間 $W$ ($k$次元) への射影演算子 $P = \sum_{i=1}^k \ket{i}\bra{i}$
- 実際，$P \ket{v} = \sum_{i=1}^k \ket{i}\braket{i|v} = \sum_{i=1}^k \braket{i|v}\ket{i} = \sum_{i=1}^k v_i\ket{i}$
:::
またこの時，下記が成り立つ
- $P^2 = P$
- $P$ は Hermite （$P = P^\dagger$）
- $Q = I - P$ は，残りの部分空間への射影演算子

### 演算子の関数
:::tip[演算子の関数]
演算子 $A$ に対して，関数 $f$ を作用させることを考える．
正規の演算子であればスペクトル分解が可能：$A = \sum_a aP_a$ （$a$：固有値，$P_a$：固有射影演算子）  
この時，$f(A) = \sum_a f(a)P_a$ と定義される．
:::

### その他
-   交換関係：$[A,B] = AB - BA$
    - 二つの演算子 $A,B$ が $[A,B] = 0$ を満たす時は可換，そうでない時は非可換という
    - 量子力学の面白さ，難しさは演算子の非可換性から来る
