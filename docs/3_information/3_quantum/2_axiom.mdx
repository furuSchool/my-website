## 結論
:::tip[量子力学の公理]
1. 量子力学は $\mathbb{C}^d$ 上の Hilbert 空間 $\mathscr{H}$ で定義される
    - 量子状態：$\ket{\psi} \in \mathscr{H}$：単位ベクトル（$\braket{\psi|\psi} = 1$）
    - 物理量 $A$：$\mathscr{H}$ 上の Hermite 演算子
    - 測定値：Hermite 演算子の固有値で与えられる（物理量 $A$ の固有値がランダムに観測される）
2. 時間発展：閉じた量子系は Schrödinger 方程式に従う：$i\hbar \frac{d}{dx}\ket{\psi} = H\ket{\psi}$ 
    - 全てのユニタリに対応するハミルトニアン $H$ が存在し，量子系の状態変化はユニタリ変換によって記述できる
:::

### Born の確率規則
:::tip[ボルンの確率規則]
$Pr(A = a~|\ket{\psi})=\bra{\psi}P_a\ket{\psi}$：状態 $\ket{\psi}$ に対し，物理量 $A$ を測定する時，測定値が固有値 $a$ となる確率
:::

-   $P_a = \ket{\phi_a}\bra{\phi_a}$：固有値 $a$ に対応する固有射影演算子。（$\ket{\phi_a}$ は固有値 $a$ に対応する固有ベクトル）
    -   $A = \sum_a aP_a$
    -   縮退がある場合，$P_a$ は 1 次元でない = 対応する固有ベクトル $\ket{\phi_{a,i}}$ が複数あり，$P_a = \sum_i \ket{\phi_{a,i}}\bra{\phi_{a,i}}$
-   測定されうるのは，物理量 $A$ の固有値のみ：$\sum_a \bra{\psi}P_a\ket{\psi} = I$ であり，これは $\sum_a P_a = I$ から導かれる

### 補足

-   全ての単位ベクトルに対応する量子状態が存在し，すべてのエルミート演算子に対応する物理量が存在すると仮定する
    -   なので，単位ベクトルならすべて量子状態だし，エルミート演算子ならすべて物理量だよ
    -   しかも，物理量 $A$ はエルミート演算子だから，必ず $A = \sum_a aP_a$ と書けるよね
-   物理量の測定値は実数だから，期待値や分散を考えられる
    -   測定するたびに，物理量 $A$ に対応する固有値(測定値)がランダムに選ばれるから
    -   期待値：$\mathbb{E}_\psi[A] = \sum_a a~Pr(A = a~|\ket{\psi}) = \bra{\psi}P_a\ket{\psi} = \bra{\psi}A\ket{\psi}$
    -   分散：$\mathbb{V}_\psi[A] = \sum (a - \mathbb{E}_\psi[A])^2~Pr(A = a~|\ket{\psi}) = \bra{\psi}A^2\ket{\psi} - \bra{\psi}A\ket{\psi}^2
     = \mathbb{E}_\psi[A^2] - \mathbb{E}_\psi[A]^2$
    -   $\Delta_\psi [A] = \sqrt{\mathbb{V}_\psi[A]}$

## Robertson の不確定性関係

：Heisenberg の不確定性関係 $\Delta_\psi[x] \Delta_\psi[p] \ge \frac{\hbar}{2}$ の一般化
:::tip[ロバートソンの不確定性関係]
$A,~B$ を任意の物理量（ = 任意のエルミート演算子），$\ket{\psi}$ を任意の状態（ = 任意の単位ベクトル）とする。  
このとき，$\Delta_\psi[A] \Delta_\psi[B] \ge \frac{1}{2} \left| \bra{\psi} [A, B] \ket{\psi} \right|$
:::

- コーシーシュワルツの不等式：$\left| \braket{\psi|\phi} \right|^2 \le \braket{\psi|\psi} \braket{\phi|\phi}$
で等号成立条件は $\ket{\psi}$ と $\ket{\phi}$ が一次従属のとき
- これから不確定性関係を証明できる

### 命題
:::tip[命題 1]
可換な物理量 $A,B$ すなわち，$[A,B]=AB-BA=0$ の時，物理量 $A,B$ は同時に測定することができる.
-   $\psi$ を $A,B$ で同時測定した場合の同時確率分布：$Pr(A = a, B = b~|\ket{\psi}) = \bra{\psi}P_aQ_b\ket{\psi}$
-   $A = \sum_a aP_a,~B = \sum_b bQ_b$ としている（スペクトル分解）
:::

また，同時対角化可能なとき，その時に限り二つの演算子は可換である.
- 同時対角化可能 $\Leftrightarrow$ $A = \sum_i a_i \ket{i}\bra{i},~B = \sum_i b_i \ket{i}\bra{i}$ と表せる
- 同じ正規直交基底 $\ket{i}$ で貼られる領域というイメージ

:::tip[命題 2]
$A$ を物理量（ = エルミート演算子），$f$ を実関数とする。この時，  
物理量 $f(A)$ の測定 = **物理量 $A$ を測定して，測定値 $a$ が得られたとき，$f(a)$ を出力する測定**
:::
これは，例えば物理量 $A^2$ を直接測定する代わりに，物理量 $A$ を測定してからその測定値 $a$ を二乗しても良いことを意味する。
