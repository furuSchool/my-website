title: 高校時代に作った数学問題まとめ
---

ここでは，高校生の時に作成しまくっていた数学の問題をまとめています。200問以上あります.
大学受験（京大や東大）で出そうな問題が作りたいと思って作っていたので，レベル感的にはそれくらいのものです。  
問題と解答の確認は結構していた（気がする）ので間違いは少ないと思いますが，あったらすみません。
解答も全てあるのですが，移すのが面倒なので，一旦は問題だけ載せます.
