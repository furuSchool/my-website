## 基本問題

:::tip[問題1 $_⭕$ ]
この解答には間違いがある。その間違いを指摘し、正しい答えを下の解答の正しい部分を用いて求めなさい。

（問）$y=ax$ と $y=x^2-2$ で囲まれる図形の面積 $S$ の最小値とその時の $S$ の値を求めよ。

（解答）$y=ax$ と $y=x^2-2$ の交点について、
$$
ax = x^2-2 \\
x^2-ax-2=0
$$
この解を $\alpha, \beta~(\alpha < \beta,~\alpha,~\beta \in \mathbb{R})$ とおく。
また、$f(x) = x^2 - ax - 2,~F(x) = \frac{1}{3}x^3-\frac{1}{2}ax^2 - 2x$ とおく。 
この時、
$$
S = \int_{\alpha}^{\beta} \{ax-(x^2-2)\}~dx = -\int_{\alpha}^{\beta} f(x)~dx 
= -\left[ F(x) \right]_{\alpha}^{\beta} = F(\alpha) - F(\beta)
$$ 

よって、 $S^{\prime} = F^{\prime}(\alpha) - F^{\prime}(\beta) = f(\alpha) - f(\beta) = 0$  
ゆえに、$S$ は常に一定となる。例えば、 $a=0$ のとき、 $S = -\int_{-\sqrt 2}^{\sqrt 2}~(x^2-2)~dx = \frac{8\sqrt 2}{3}$ より、
すべての $a$ で $S = \frac{8\sqrt 2}{3}$ となる。
:::
