---
title: 自転車で東京から兵庫まで行った話
---

大学3年生の夏休みに、せっかくだし自転車で東京から実家の兵庫まで帰省しようと思い、9/6〜9/10 で帰った。巣鴨あたりから、夙川あたりまで。
1号線を中心に進み、車しか通れないところを違う道で通った。東海道で日本橋から京都へ行こうみたいな人と同じようなルートを通ったと思う。

- https://oh-tsuji-kaitai.com/cyari-tokaido-intro/

## 1日目

東京からスタート。10:00くらい出発。初めは順調で、戸塚のまつのやでご飯を食べる。だったが、茅ヶ崎市あたりでしんどくなってきた。小田原までは半分死んでたけど、なんとか到着。

なんと、前日の自分は三島のホテルを予約していたが、明らかに無理じゃね、と感じ始める。

17:00くらいに小田原について、そこから箱根を登ろうとするが、そもそも前の太ももが痛すぎて、全然こげない。歩くことによって登ることを決める。

しばらくはなんとかなっていたものの、18時を過ぎると急激に暗くなる。車も通るし、明らかに危ないので、大平台あたりで、近場に泊まることを決意。
ホテル代はキャンセル料全額取られるけど……。しばらくは、そのまま登山していた。

しばらく登山していると、19:00くらいに小湧園あたりにつく。ホテルや旅館が多く、その中のゲストハウスを予約。
コンビニで買い出しして、そのゲストハウスで一泊。外国人さんも多く泊まるゲストハウスで、フロントがバーになっていた。
なんでかはわからないが、結構楽しかった。ブリスベンから来たポリスオフィサーと少しだけ話した。

## 2日目

9:00過ぎくらいに出立。箱根の登山をまた行い、10時過ぎくらいに1号線の最高点に到達。そこから元箱根（芦ノ湖あたり）までは坂道をくだるだけで、めっちゃ気持ちよかった。
その後、ちょっと登って箱根峠に着いたあと、三島中心部まで0コギで坂道を下る。自転車乗ってきて一番気持ちよかった瞬間かも。40km/hくらい出てた気がする。
途中、三島スカイウォークに会ったが、1300円と高いし、時間かかりそうだし、別におもんなさそうだったから写真のみ撮って帰った。

三島に着いたのは12時前くらい。やっぱり、最初三島でホテルを取っていたのは完全に目測ミス。絶対に無理。

その後、富士市までほぼ何も考えずいく。信号ないし舗装されてるしで通りやすすぎた。時速17kmくらい出てた気がするが、だいぶ疲れた。
富士市のさわやかハンバーグを食べようとしたが、100分待ちだった。現在13時。
めちゃ疲れていたので、近くのミニストップのイートインスペースでクーリッシュを食べながら時間潰し。その後は、店内の待合席で本を読んで時間潰し。結果的に、120分以上かかった。
人生2回目のげんこつハンバーグを食べたが、やっぱりめちゃくちゃ美味しかった。

その後、焼津のホテルを予約。静岡駅までは、[サイト](https://oh-tsuji-kaitai.com/cyari-tokaido-intro/)を参考に移動。
途中、由比周りで軽い登山（峠越え）をおこなったが、思ったより短かったため、それほど疲れなかった。
18:00ごろに静岡駅につき、そこから焼津駅に向かったが、土砂崩れで車両が通行できないことが発覚。焼津にいくには1号線で迂回しないといけないことがわかったので、
その手前の用宗駅に自転車を止め、電車を使って身一つで焼津駅に到着する。そこからはホテルで休憩タイム。

:::note[その他書き残し]
- 思いついた水平思考ゲーム『男は、自転車で東京から兵庫まで行こうとしているのに、電車を使った。その理由は？』
- 東海地方は、感知式じゃなくて感応式みたい。なんで？
:::

## 3日目
9時過ぎごろに出立。用宗駅から出発し、1号線で山を越える。
途中、1号線のトンネルを通れないと勘違いして、他の道のトンネルを通ろうとしたが、これも土砂崩れで通行止めでだいぶ焦った。全然国道1号線通れた。
その後は、道なり。以下、静岡市 →（焼津市）→ 藤枝市 → 島田市 → 大井川 → 金谷 から、掛川へ山越え。
金谷まで結構遠く、山越えは 200m くらいでそこそこ大変だった。やっぱ山を下る時が一番気持ちいい。

13時半くらいに、掛川でうな重を食べる。意外と掛川市は歴史的っぽい雰囲気。
うなぎって、値段の割にはめっちゃ美味いわけではないよね。おいしいけど、コスパは全然あってない。個人的コスパ最弱はマツタケ。間違いない

ご飯食べてめちゃ疲れたが、漕げるうちにできるだけ行こうとする。
その後は、袋井市 → 磐田市（ここまでは結構すぐ）→ 天竜川→ 浜松市（めちゃでかい）。
浜松市内で、シャトレーゼによって抹茶アイスを食べる。浜名湖を南に回るが、浜松市街から浜名湖まで遠かった。

その後は、湖西市 → 山越え（100m くらい）→ 豊橋市 で到着。最後の方はなぜか調子が良かった。19時くらいにはつけた。ドーミーに泊まったがめえっちゃよかった。

:::note[その他書き残し]
- 豊橋市中心部一歩手前くらいで、ものすごい量のカラスが電線中に止まっていた。しかも、無数のカラスがそこに向かって集まってた。怖かった。
- 静岡は、内陸もある程度栄えてる感じだった。山の間にそこそこの街がある感じ。あと、静岡市周りの海岸は山も多め。
:::


## 4日目
8:30 くらいに出立。めちゃくちゃ調子が良かった。なんでやろうな、と思ったら、日が出てないからだった（と思う）。
豊橋 → 岡崎は100mくらいをゆるやかに超えて、まあまあ遠かったけど全然すぐについた。朝でトラックが多かったのが危なかった。

そのあとは、豊橋市 → 豊川市 → 岡崎市 → 安城市 → 知立市 → 刈谷市 → 豊明市 → 鳴海（名古屋市）→ 熱田（名古屋市）

ずっとザ中都市という感じ、、、。岡崎市街は車と信号多くて道ガタガタで走りにくかったが、それ以外は基本走り良かった。

熱田神宮で中吉をひき、きしめんを食べ、そこから出発。おみくじは、自分でガシャガシャしてからその番号を伝える方式だった。
嘘の番号も全然行けそう。熱田神宮は思ったよりはパッとしなかったが、あまり観光地化されていないという点ではいいのかもしれない。

そこからも1号線をひたすら進み、途中休憩を挟みつつ、
名古屋市（やっぱりでかい）→ 蟹江町 → 弥富市 → 木曽川 → 桑名市 →（途中）長良川・揖斐川 → 朝日町 → 川越町（記憶ない）→ 四日市市 → 鈴鹿市 → 亀山市
明日の雨が心配だったので、できるだけ滋賀県に近づくため、亀山駅の次の関駅に自転車を置き、電車を使って亀山駅へ戻る。

亀山駅で1800円くらいのトンカツを食べる。自分一人のために小皿やらタレやら15個くらい並んですごく豪華だった。
個人的に、コスパ最強外食はトンカツだと思う。（上限がそこそこ広く、かつ満足度が高い。）

:::note[その他書き残し]
- 名古屋市にも港区があった。どこにでもあるなあ。
- 名古屋と三重の間はめっちゃ川が多い。
- 木曽三川めっちゃ忘れてた。大井川や天竜川と違って、ただ広いだけでなくて、水量もすごいから涼しかったし怖かった。
    - 木曽川：源流は針盛山、全国７位の長さ、木曽→中津川→美濃加茂/可児で飛騨川と合流
    - 長良川：木曽川水系らしい、大日ヶ岳が源流、郡上市のさらに北から線路に沿って流れてきているイメージ
    - 揖斐川：木曽川水系らしい、冠山が源流、大垣のさらに北からきているイメージ
- 四日市〜亀山はほぼ高速道路だった。車もそれくらいのスピードでくるし、普通に危険
- 四日市は意外と最近発展してそうな街だった。
- 亀山も、駅はボロいのにその周りは新興住宅街の雰囲気で、なんでも揃う。少し変な街。
:::

## 5日目
9時10分くらいの電車に乗車、関駅に到着。雨が降っていたので、前日に買った折り畳み傘をさしつつ歩いて山登り。
山に南寄りの風が当たって雨が降っているようで、これからも振り続けそう、峠まで10kmないくらいだったので、歩いてもなんとかなるだろうという判断。

途中、雨が弱くなったら自転車に乗りつつ、300m ちょっとの鈴鹿峠を越える。
その後、トンネルを抜けるとばったり晴れたので、坂道を下りつつ、土山 → 水口 → 湖南市 → 栗東 といき、栗東と草津の間ではなまるうどんを食べる。
調子がすこぶる良かった。自転車のコツを掴んできた感じ。

:::note[詳細]
- 前日に関駅に自転車をおいたのはまじでナイス判断。歩き時間1時間くらい短縮できたと思う。
- 滋賀に早めに抜けようとしたのも良い判断。
- 水口〜大津は小・中都市という感じ。水口が、その割に最近できたチェーン店とか施設が多そうだった。30年後にはさびれてそう。
- 基本的には走りやすかった。曇ってたし、信号少なめで道路は舗装されている。
- 甲賀あたりはほぼ高速道路だった。しかも通れるところと通れないところがあってややこしい。
- 甲賀（こうが）
:::

その後は、草津市 → 近江大橋 →（膳所）→ 大津市 →（逢坂）→ 山科 → 京都市 → 久御山町 → 八幡市 → 枚方市 →
 寝屋川市 → 守口市 → 京橋 → 梅田 → 福島 → 尼崎市 → 西宮市
で到着。基本1号線→2号線

距離はあるはずだが、調子が良かったのと、ゴールが近かったのでそんなに疲れなかった。20時自宅着。

:::note[詳細]
- 近江大橋はなんとも言えない。長いだけ感。
- 山科は200mくらいの山越えだが、傾斜が急できつかった。2回登った。が、上りよりも下りの方が明らかに高さがあり、琵琶湖の高さを感じた。
淀川流れてるもんね。水面85mくらいらしい。
- 膳所は大津の中心付近にあって驚き。歴史的な雰囲気。
- 京都市はやっぱりすごい。今と昔がうまく調和してて、街として完成している感がある。
- 国道1号線は五条、JRは八条
- 伏見区あたりで雨第1弾に遭遇。そこまで降らなかったが、ここで、雨雲レーダーをみて大阪がやばいことに気づく。
とりあえず、行けるところまで行く。ちょうど100mくらい登って京都から大阪へ入ったイミングで大雨に遭う。
ひとまず枚方のニトリモールに避難。雨雲レーダーを見つつ、雨のときは歩いて進み、雨が弱い時に自転車に乗ることを決意。
- ただ、枚方での雨は少し進むとすぐマシになって、そこから京橋くらいまでは問題なく進めた。運が良い。
- 京橋をちょっと行ったタイミングで、雨第3弾、急なゲリラ。雨雲レーダーにない雲が突然出現。
こういう時は風で流すだけの雨雲レーダーは全く頼りにならないので、リアルタイムの解析雨量を見つつ、バス停で雨宿り→コンビニに移動。
やみそうになかった&尼崎は大丈夫そうだったので、大雨で傘歩き、弱雨で自転車を繰り返すことに決める。
- 福島すぎるくらいまではマジで大雨。道路冠水してたし、やばかった。靴は浸水。リュックも浸水。
ただ、淀川を渡ったくらいから雨がマシになり、そこから自転車に乗れるように。結果としては予定時刻通り（20:00）くらいにつけた。
- 171でいけば雨にあわんかったかもなあ。
- なじみがあるのは、吹田市 → 摂津市 → 茨木市 → 高槻市 → 島本町 → 大山崎町 → 長岡京市 → 向日市 の阪急ルートだったので、京阪ルートで新鮮だった。
- 京都〜大阪でなぜか国道1号線が2本ある。いみがわからん。
:::

## まとめ
- アクシデントはありつつも、そんなに大事件をなく、足が死んだりすることもなく、意外といけた。箱根が一番きつかった。
- 雨が最終日以外降らなかったのが神ががっていた。
- １〜３日目は平気で35度とか言ってた。9月なのに。意味わからん。

:::note[その他書き残しなど]
- コンクリートは、粒大きめのやつと粒小さめのサラサラのやつとかある。車が出入りするところがガタガタの確率が高い。
あと、白線がわざとがたがたな道路もたまにある。車がはみ出さないようにだろうが、自転車にとってはめちゃくちゃ邪魔。
- 山道は18時には出るべき（9月なら）
- 普通の道でも、超都会か知っている道でない限り、18:30までには出たい。
- 昼前が一番しんどくなる。ご飯前かつ暑くなってくるため。それ乗り越えたら意外となんとかなる。
- 200m くらいの山ならまあなんとか行ける。それ以上は普通にしんどい。
- 新幹線が近くにある道も結構通ったが、いちいちかっこいい。音といいその速さといい。これに乗ればいいだけやのに、と思っていた。
- NAVITIME の 13km/h なら、休憩やごはんを合わせてちょうどくらいで見積もれる。
- Google map の自転車はやっぱり早い。特に都内なんか全然当てにならない。けど、信号ない走り良いところならちょうどくらい？
- Google map は、直線で良い道でもわざわざ違う道を提示することがよくある。
Google map は30分くらいで着く時に調べるよう、NAVITIMEはそれ以上で調べる時に使う、みたいに使い分けができそう。
Google map は、中継地点みたいのをいくつか設定してルートを計算するアルゴリズムなのか？

- トンネルの前後は上りto下り（そらそうだけど新たな発見）
